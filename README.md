# ToDoアプリ

## GitHub Page
https://sae-github.github.io/to-do-app/

## 実装した経緯

Redux、TypeScriptの理解を深めるため。

ReduxToolkitを使用した状態管理や、TypeScriptの基本的な書き方等を学ぶため。

## 使用した言語/フレームワーク...等

- React
- TypeScript
- ReduxToolkit
- TailwindCSS
- Material Icons

## 機能

- ダークモードへの切り替え
- タスクの追加・削除
- フィルタリング機能(完了/未完/全て)
- Drag＆Drop
